HTML 块状元素和行内元素

发表于 2019-03-11 分类于 HTML/CSS/JavaScript

HTML 中有块状元素也有行内元素，即使如此，也能通过 CSS 的 display 改变元素的默认效果。如使用 display: block 使得元素显示为块状元素效果，使用 display: inline 则可以使得元素显示为行内元素效果。

使用以下代码可得到默认的块状元素和行内元素：

<body>
